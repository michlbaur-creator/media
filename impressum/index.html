Datenschutz

Media Mibaso ist eine reine Web-App ohne Nutzerkonto. Es werden keine personenbezogenen Daten erhoben, gespeichert oder an Dritte weitergegeben. Es gibt kein Tracking und keine Werbung.

Der geschützte Kursbereich ist durch ein gemeinsames Passwort gesichert; es gibt keine persönlichen Konten. Kleine technische Einstellungen (z. B. ob die Willkommens-Tour schon angezeigt wurde) werden ausschließlich lokal auf Ihrem Gerät im Browser abgelegt und verlassen es nicht. Sie können diese jederzeit über die Website-Daten Ihres Browsers löschen.

Verlinkte Materialien (PDFs, Videos, Quizze, Flipbook) liegen teils auf anderen Diensten (mibaso.de, YouTube, Publuu, Revisely, Microsoft Forms). Beim Öffnen gelten die Datenschutzbestimmungen des jeweiligen Anbieters.

Die Seite wird über GitHub Pages bereitgestellt. Beim Aufruf können technisch bedingt Server-Logdaten (z. B. die IP-Adresse) durch den Hosting-Anbieter verarbeitet werden; darauf habe ich keinen Einfluss.
